# AGENTS.md - WB Data Analyzer Workspace

## 🎯 项目定位

这是 **World Bank Data360 Analyzer** 的独立工作空间。

## 🚀 快速开始

```bash
npm install
npm test
```

## 📊 核心功能

- 数据查询 - WDI 数据库
- 自然语言 - 中英文查询
- 计算模块 - 增速、CAGR、指数化
- IMF WEO 风格 - HTML 输出
- 导出功能 - CSV/PNG

## 🌍 预置群组

- BRICS: `brics_founding`, `brics`
- NDB: `ndb_members`
- 收入分组：`high_income`, `upper_middle_income`
- 地域分组：`asia`, `europe`, `africa`
